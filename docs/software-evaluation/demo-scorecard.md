## Demo Questions

- Show how a Sunday offering is entered and deposited.
- Show how a designated fund gift is tracked.
- Show how a reimbursement or expense is entered.
- Show how receipts can be attached or linked.
- Show monthly budget-to-actual reports.
- Show bank reconciliation.
- Show permission settings for Treasurer, Bookkeeper, Pastor, and Finance Committee.
- Show year-end giving statements.
- Show export options.
- Explain migration process and support.
