# Software Requirements

Status: Draft

## Must Have

- Fund/account tracking.
- Budget reporting.
- Contribution records.
- User permissions appropriate for church roles.
- Exportable financial reports.
- Data backup and recovery.

## Should Have

- Reimbursement workflow support.
- Document attachments.
- Role-specific dashboards.
- Import tools for existing data.

## Questions

- What system is currently used?
- What data must be migrated?
- Who needs access?
- What reports are required for committees and church business meetings?
