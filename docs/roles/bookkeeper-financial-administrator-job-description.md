# Bookkeeper / Financial Administrator Job Description

Status: Draft

## Position Summary

The bookkeeper / financial administrator supports day-to-day financial operations, records transactions, prepares reports, and maintains documentation according to church policy.

## Responsibilities

- Record income and expenses.
- Process bills and reimbursements according to approval policy.
- Maintain bookkeeping records and supporting documentation.
- Prepare monthly reports for review.
- Support payroll processing if assigned.
- Coordinate with the treasurer and finance committee.
