# Source Materials

This folder contains copies of existing documents used to understand current practices, prior proposals, role definitions, and governance context.

Because this repository has been published through GitHub Pages, source materials in this folder should be treated as public unless repository and Pages settings are intentionally changed and verified.

Binary files such as PDFs, DOCX files, and PPTX files cannot be fully checked by the public-content audit. Review them manually before broad public sharing.

## Import Policy

Imported here:

- Bylaws and governance reference material.
- Existing job descriptions and role documents.
- Blank job application forms.
- Prior proposals related to finance, bookkeeping, treasurer duties, personnel structure, or accounting software.
- Software and conversion notes that may affect financial operations.

Intentionally not imported unless later approved:

- Actual candidate applications.
- Completed reference checks or evaluations.
- Background-check account walkthroughs or credentials.
- Deacon family lists or member rosters.
- Security, children ministry, or unrelated ministry program files.

Use active Markdown files in `../docs/` for analysis and recommendations. Keep source files intact.

Before adding more source files, confirm that they do not contain donor records, payroll details, bank account numbers, passwords, Social Security numbers, actual candidate applications, reference-check notes, background-check results, confidential personnel issues, or private financial data.
