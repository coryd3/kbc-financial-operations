# Review Intake

This folder is reserved for local, temporary committee-review intake.

Returned Word documents and converted review files may contain comments or information that is not appropriate for the public repository. The contents of `review-intake/finance/` are ignored by Git. Do not force-add them.

Use `python scripts/ingest_finance_review.py <returned-folder-or-zip>` to create a review workspace. Apply accepted changes deliberately to the canonical Markdown files under `docs/`.
