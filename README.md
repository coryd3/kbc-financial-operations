# KBC Financial Operations

This repository organizes source materials, working documents, policies, procedures, templates, and decision records for Kingsville Baptist Church financial operations.

The goal is to produce clear, maintainable materials for finance governance, bookkeeping administration, committee responsibilities, software evaluation, and church communication.

## Repository Layout

- `source-materials/` - Existing reference materials copied from prior church documents.
- `docs/` - Active working documents for assessment, decisions, policies, procedures, roles, and communications.
- `templates/` - Reusable formats for motions, agendas, recommendations, policies, and job descriptions.
- `archive/` - Superseded drafts and historical working material that should no longer be treated as current.

## Working Norms

- Treat this repository as private unless the finance/personnel leadership explicitly approves otherwise.
- Keep personally identifiable information, candidate applications, reference checks, financial account credentials, and member lists out of version control.
- Prefer Markdown for active drafting.
- Keep source documents intact and use working docs to summarize, compare, and propose changes.
- Record meaningful decisions in `docs/02-decision-log.md`.
